# Café Dourado - Landing Page ☕✨

Uma landing page moderna, responsiva e elegante desenvolvida para a cafeteria fictícia **Café Dourado**. Este projeto visa demonstrar habilidades em desenvolvimento web front-end utilizando HTML5, CSS3 e JavaScript Vanilla (sem frameworks), proporcionando uma experiência de usuário fluida e visualmente atraente.

## 🌟 Funcionalidades

- **Design Responsivo:** A página se adapta perfeitamente a diferentes tamanhos de tela (desktop, tablet e mobile).
- **Navegação Suave (Smooth Scrolling):** Rolagem suave ao clicar nos links do menu para as respectivas seções.
- **Cabeçalho Dinâmico:** O cabeçalho (navbar) altera seu estilo e ganha uma sombra/fundo translúcido ao rolar a página para baixo.
- **Menu Mobile:** Menu de navegação tipo hambúrguer para dispositivos móveis.
- **Filtro de Cardápio:** Sistema interativo de filtragem para visualizar as especialidades (Todos, Cafés, Bebidas Geladas, Acompanhamentos) sem recarregar a página.
- **Animações ao Rolar (Scroll Reveal):** Elementos da página surgem suavemente conforme o usuário rola para baixo (implementado com `IntersectionObserver`).
- **Formulário de Contato com Modal:** Simulação de envio de formulário de contato com feedback visual (modal de sucesso) para o usuário.
- **Ícones Lucide:** Utilização da biblioteca leve e moderna de ícones [Lucide Icons](https://lucide.dev/).

## 💻 Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Estilização avançada, CSS Grid, Flexbox, transições, animações customizadas e variáveis CSS para facilitar a manutenção do tema.
- **JavaScript (ES6+):** Lógica de interação no lado do cliente (filtros, modais, menu mobile, observer para animações).

## 🚀 Como Executar o Projeto

Como o projeto é estático e desenvolvido apenas com tecnologias front-end puras (HTML, CSS, JS), não é necessário um servidor complexo ou banco de dados para rodá-lo localmente.

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Leonnardo21/cafe-dourado-landing-page.git
   ```

2. **Navegue até o diretório do projeto:**
   ```bash
   cd cafe-dourado-landing-page
   ```

3. **Abra o projeto no seu navegador:**
   - Você pode simplesmente dar um clique duplo no arquivo `index.html`.
   - Alternativamente, para uma melhor experiência com requisições e evitar possíveis problemas com CORS (caso você expanda o projeto futuramente), você pode usar a extensão **Live Server** no VS Code ou outro servidor local de sua preferência.
